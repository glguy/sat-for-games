Description

Symbolic selection from a finite list of alternatives. This module allows using normal Haskell values symbolically.

We can extract the selected value using either runSelect or decode.

Select values can be symbolically compared using comparisons of their concrete possibilities. For example (===) will use (==) on the underlying choices.

>>> :{
solve $
  do let letters = 'a':|"bcdefg"
     x <- selectList letters
     y <- selectList letters
     assert (pure 'c' <? x && x <? y && y <? pure 'f')
     return (x,y)
:}
Just ('d','e')
