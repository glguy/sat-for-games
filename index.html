Introduction to using SAT to solve games.

This package provides the Ersatz.Prelude module as a convenient way to import most of the things you'll need while using Ersatz.

The LightsOut module provides a simpler example of a game that we can solve using a SAT solver. This game only requires us to xor the right elements from a grid together.

The Zhed module provides a more complex example of a game that we can solve using a SAT solver. This game will take us through counting, symbolic choice from a map, and symbolic choice of normal, concrete Haskell values.

The TotalMap module provides a useful abstraction built on top of the standard Data.Map module. This will allow us to attach a default value to a map.

The Select module will allow us to make symbolic choices from a list of values. This will help us to avoid having to make many one-off Codec and Variable instances.

The Choice module will provide an abstraction for types that support symbolic choice between two alternatives.
